## Python input function
``` python
print("Hello" + input("What's your name."))
```

執行上方程式碼時，`input("What's your name.")` 會被替換成輸入的值，並和Hello組成字串，print出來。可透過Thonny 這個開發 IDE 一個步驟一個步驟看。
