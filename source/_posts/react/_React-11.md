Redux 是實作狀態管理機制的一個套件，不只能用在 React，其概念是從 Flux 延伸而來。

### 狀態管理

* 狀態管理對於前端特別重要，如：資料跟前端畫面的對應關係、如何管理資料、不同 Component 共用相同資料等。
* React 只需要管資料。
* Flux 改變資料狀態流程： View -> Action -> Dispatch -> Store -> View(單向資料流)
