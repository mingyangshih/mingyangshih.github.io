小結

Webpack 是透過 plugins 屬性來設定需要使用的外掛程式清單的， plugins 屬性是一個陣列，裡面的每一項都是外掛程式的 instance，在產生 instance 時可以透過建置函數傳入這個元件支援的設定屬性。
如：filename 屬性，可告訴外掛程式 CSS 檔名希望產生的格式。
